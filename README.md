## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/generalpooria/qa-automation-saucedemo.git
   cd qa-automation-saucedemo
   ```
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
3. Run tests:

   ```bash
   pytest -v
   ```
